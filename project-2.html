Smart Campus Management System

A full-stack web application to manage students, teachers, class schedules, and events. Features include role-based access, attendance tracking, and admin dashboard. Built using Spring Boot,React, HTML, CSS, and MySQL.

Live Link

Project Overview

This project was developed during my MCA academic journey with the goal of solving a real-world problem through a web-based application. The project focuses on providing a user-friendly and efficient solution that allows users to perform specific tasks with ease.

The application was built using Java, Spring Boot for the backend, and HTML, CSS, and JavaScript for the frontend. It includes features such as user authentication, data storage with MySQL, and RESTful APIs for communication between the frontend and backend. The project follows the MVC architecture, ensuring a clean and maintainable codebase.

Through this project, I enhanced my skills in full-stack development, learned how to manage project requirements, and improved my ability to debug and test effectively. It was a valuable learning experience that helped me understand how different technologies work together to build a complete software solution.
