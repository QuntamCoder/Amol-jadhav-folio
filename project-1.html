HR Management System

A web-based system to manage employee records, leave tracking, and payroll. Built with Java, JSP, HTML, CSS, JavaScript, and MySQL. Includes login roles for Admin and Employees, data validation, and CRUD operations.

Live Link

Project Overview

This project was developed as a part of my academic curriculum during my MCA program. The main objective of the project was to create a fully functional web application that solves a real-world problem by using modern development technologies.

The application includes features such as user authentication, dynamic content rendering, and data storage using a relational database. It follows the MVC architecture and was built using technologies like Java, Spring Boot, HTML, CSS, JavaScript, and MySQL. The goal was to ensure a responsive and user-friendly experience across different devices.

This project helped me understand full-stack development better, including backend logic, frontend design, RESTful APIs, and database integration. It also taught me how to manage tasks independently, test thoroughly, and deploy applications effectively.
